A lightweight Python-based chatbot that uses conditional logic to interact with users. This project demonstrates fundamental programming concepts like loops, functions, and control flow.

Features
Persistent Conversation: Uses a while loop to keep the chat active.

Case Insensitivity: Automatically converts user input to lowercase to ensure matches are found regardless of capitalization.

Graceful Exit: Users can end the session simply by saying "bye".

Fallback Logic: Includes a default response for unrecognized commands.

Concepts Covered
Functions: The logic is encapsulated in a chatbot() function.

Input/Output: Interaction via the standard terminal.

Conditional Statements: if-elif-else structures to determine responses.

Loops: A while loop to manage the application lifecycle.

Chatbot Response
"Hello"	"Hi!"
"How are you"	"I'm fine, thanks!"
"What's up?"	"I'm sorry, I don't understand that yet."
"Bye"	"Goodbye!" (Program exits)
